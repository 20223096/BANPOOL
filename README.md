# BANPOOL (금지어 수영장 파티)

Next.js + Socket.IO 실시간 파티 게임 MVP.

## 설치 & 실행

```bash
cd street_food
npm install
npm run dev
```

- 웹: http://localhost:3000  
- API: http://localhost:4000 (`GET /health`)

`web/.env.example` → `web/.env.local` (소켓 URL 변경 시)
